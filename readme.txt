# Markit Media Group Website

## Overview
Welcome to the Markit Media Group website project! This repository contains a modern, responsive website for **Markit Media Group**, an Indian digital brand and marketing company specializing in content creation, social media management, and digital marketing services. 

The website adheres to Indian digital and social media regulations, including:
- **Information Technology Act, 2000**
- **IT (Intermediary Guidelines and Digital Media Ethics Code) Rules, 2021**
- **Digital Personal Data Protection Act, 2023 (DPDPA)**

The site features a sleek, **dark-themed design** with **blurry-transparent effects, cyan accents, and white text** for readability. It includes multiple pages built using **HTML5, CSS3, and Vanilla JavaScript**, ensuring a seamless user experience across devices.

---

## Features
- **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices using CSS3 media queries.
- **Dark Theme:** Consistent dark background (`#1a1a1a`) with white text and cyan accents (`#00b4d8`) for a modern look.
- **Blurry-Transparent Effects:** Utilizes `backdrop-filter: blur(10px)` for a frosted-glass aesthetic on key sections.
- **Navigation:** Fixed navbar with smooth opacity transitions, hover effects, and dot indicators for active states.
- **Legal Compliance:** Includes Terms and Conditions and Privacy Policy pages adhering to Indian laws.
- **Interactivity:** Simple scroll-based animations and hover effects powered by Vanilla JavaScript.
- **Accessibility:** Semantic HTML5 structure and ARIA attributes for better accessibility.

---

## Pages
- **Home:** Features the Markit Media Group logo, slogan ("We Make Something Creative"), and an introduction to services (`index.html`).
- **Privacy Policy:** Details privacy practices compliant with **DPDPA 2023** (`privacy.html`).
- **Terms and Conditions:** Legal agreement outlining usage rules, compliant with Indian regulations (`tas.html`).

---

## Technologies Used
- **HTML5:** Semantic markup for structure and accessibility.
- **CSS3:** Styling for responsiveness, animations, and dark-themed design (stored in `css/` and `scss/` folders).
- **Vanilla JavaScript:** Basic interactivity (e.g., scroll effects, form handling, stored in `js/` folder).

---

## Prerequisites
- A modern web browser (**Chrome, Firefox, Safari, Edge**) to view the website.
- No external dependencies or server setup required—just open the HTML files in a browser.

---

## Installation
```sh
# Clone the repository
git clone https://github.com/your-repo/markit-media.git

# Navigate to the project folder
cd markit-media

# Open any HTML file in a browser
tart index.html
```

---

## Usage
- Explore the website by navigating through the **fixed navbar** to access different pages.
- Interact with elements like the **logo, navbar, and scroll-based animations**.
- Review the **Terms and Conditions (`tas.html`)** and **Privacy Policy (`privacy.html`)** for legal and privacy details.
- Customize the content, images, or styles in the **HTML, CSS, JavaScript, and SCSS** files as needed.

---

```
MARKIT MEDIA/
```
